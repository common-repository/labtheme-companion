== Description ==

Labtheme Companion is a lightweight and safe premium plugin. The plugin generates seven custom post types (FAQs, Clients, Events, Services, Portfolios, Team, Courses and Testimonials), exclusive widgets, and some useful shortcodes.

== Installation ==
**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search Labtheme Companion > click on install or Click on upload > Select labtheme-companion.zip
3. Activate 'Labtheme Companion' from your Plugins page.
